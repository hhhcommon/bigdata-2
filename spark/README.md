## 环境准备
### 安装scala
#### 下载scala-2.11.12.tgz包并解压到目录下
```shell script
tar -zxvf scala-2.11.12.tgz -C ~/software/
```
#### 添加scala目录到PATH
```shell script
vim ~/.bash_profile

epxort PATH=$PATH:/Users/zhangzhiwen/software/scala-2.11.12/bin

source ~/.bash_profile
```

### IDEA 安装scala插件
到插件市场搜索scala插件并安装
